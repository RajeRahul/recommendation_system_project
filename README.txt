Follow the following steps to run this application.


#NOTE: make sure that this folder has filesharing access from docker desktop, if not follow these steps:
1. open docker desktop dashboard
2. go to settings
3. go to resources
4. go to filesharing
5. click on the plus sign and add the project path.

open command prompt in the project folder and type the following commands:

1. docker-compose build

2. docker-compose up

let the docker-compose up command running, do not close the command prompt.

then run the script.bat file

Now you should be able to access the web application.
